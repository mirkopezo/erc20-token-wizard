## Setup

1. Clone Repository

    ```sh
    $ git clone https://github.com/mirkopezo/erc20-token-wizard.git
    $ cd erc20-token-wizard
    ```

2. Install Dependencies

    ```sh
    $ npm install
    ```

3. Run DApp

    ```sh
    $ npm run start
    ```

    Browser will launch on [http://localhost:3000/](http://localhost:3000/).
